## âœ… API (Application Programming Interface)

**Meaning:** API allows two software systems to communicate. We send a request and the server returns data.

**In this project:** We use `NewsAPI` to fetch real-time news.

```python
# Example API request base URL
https://newsapi.org/v2/everything
```

---

## âœ… API Key

**Meaning:** A secret unique key given by API providers to authenticate users.

```python
api = "your_api_key_here"
```

Used like a password to prevent misuse of API.

---

## âœ… `requests` Module

**Purpose:** To send HTTP requests and get data from internet APIs.

```python
import requests
```

### â­ Sending GET Request

```python
r = requests.get(url)
```

This retrieves information from the given API URL.

---

## âœ… User Input in Python

Takes data from user through keyboard.

```python
query = input("Enter topic: ")
```

---

## âœ… String Formatting (f-strings)

Used to insert variables inside strings.

```python
url = f"https://newsapi.org/v2/everything?q={query}&apiKey={api}"
```

---

## âœ… JSON Data Handling

API sends data in **JSON format**. We convert it to Python dictionary using:

```python
data = r.json()
```

### Accessing JSON keys

```python
articles = data.get("articles", [])
```

`.get()` is safer because it avoids errors if key is missing.

---

## âœ… Conditional Statements

To check API status and handle errors:

```python
if data.get("status") != "ok":
    print("Error fetching news")
```

---

## âœ… Looping Through Articles

We use `for` loop to display each article.

```python
for article in articles[:10]:
    print(article.get("title"))
```

`[:10]` â†’ Shows only first 10 results.

---

## âœ… Printing Separator

For clean UI

```python
print("-" * 60)
```

Prints 60 dashes.

---

## âœ… Error Handling Logic

Case when no results found:

```python
if not articles:
    print("No articles found.")
```
